Testo dell'esercizio:

  1. Change the text of two buttons to + and - respectively.
  2. Change the color of the decrement button to black.
  3. Create an h1 element and attach it to the html body.
  4. Create a variable called count and attach it to the created h1 element.
  5. Add event listeners to + & - buttons such that when they are clicked the count increases and decreases respectively.
